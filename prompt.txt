MANUAL DO WILLIAM – LOJA DO FIADO (AFILIADOS)
Versão simples e passo a passo para o Sr. William (70 anos)

============================================================
1) O QUE ESTE SITE É (EM PALAVRAS SIMPLES)
============================================================
- Este site é uma VITRINE DE PRODUTOS.
- As pessoas entram, veem os produtos e clicam no botão.
- O botão leva direto para o site da Shopee, Amazon ou Mercado Livre.
- Aqui NÃO tem compra, NÃO tem pagamento e NÃO precisa cadastro.

Em resumo: é como um catálogo bonito que leva para a loja verdadeira.

============================================================
2) COMO O VISITANTE USA O SITE (SEM CADASTRO)
============================================================
1. Entra no site.
2. Vê os produtos em destaque.
3. Clica no botão do produto.
4. Vai direto para o site da loja (Shopee/Amazon/Mercado Livre).

============================================================
3) O QUE O SR. WILLIAM VAI FAZER (CADASTRAR PRODUTOS)
============================================================
Hoje o site NÃO tem painel de administrador pronto.
Para alterar os produtos, o cadastro é feito editando um arquivo do site.
(Se quiser, posso criar depois um painel simples de admin com senha.)

O arquivo principal onde ficam os produtos é:
client/src/pages/home.tsx

Dentro dele existe uma lista chamada MOCK_PRODUCTS.
É ali que o Sr. William vai cadastrar, trocar ou remover produtos.

============================================================
4) PASSO A PASSO NO REPLIT (MUITO SIMPLES)
============================================================
1. Abra o Replit do projeto.
2. No lado esquerdo, procure a pasta:
   client → src → pages → home.tsx
3. Clique nesse arquivo.
4. Procure por “MOCK_PRODUCTS”.
5. Cada produto é um bloco entre { }.
6. Para adicionar produto:
   - Copie um bloco existente.
   - Cole abaixo.
   - Troque o título, preço, etc.
7. Salve o arquivo.
8. Volte para a aba do site e atualize (F5).
9. Pronto: o produto aparece.

============================================================
5) CAMPOS DE CADA PRODUTO (O QUE SIGNIFICA)
============================================================
Cada produto tem estes campos:

- id: um código simples (ex: "p9", "p10").
- title: nome do produto.
- price: preço atual.
- oldPrice: preço antigo (opcional).
- installment: texto de parcelamento (ex: "10x sem juros").
- rating: nota do produto (ex: 4.7).
- reviews: quantidade de avaliações (ex: 120).
- tag: etiqueta (“OFERTA”, “DESTAQUE” ou “FRETE GRÁTIS”).
- category: categoria (tem que ser uma das que já existem).

Categorias válidas:
Destaques, Eletrônicos, Casa, Beleza, Moda, Mercado, Infantil, Ferramentas.

============================================================
6) COMO COLOCAR O LINK DE AFILIADO
============================================================
Hoje o botão “Adicionar” NÃO leva para lugar nenhum.
Para virar link de afiliado, precisamos adicionar um campo novo.

Sugestão simples (para fazer depois):
- Adicionar a chave "url" em cada produto, por exemplo:
  url: "https://www.amazon.com.br/seu-link"

Depois trocar o botão para abrir esse link.

Eu posso fazer isso para você. Se quiser, eu altero o código e deixo pronto.
